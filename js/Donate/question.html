Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents (like HTML). It represents the structure of a webpage as a tree of objects, where each HTML element is a node. JavaScript can interact with the DOM to dynamically update or change the content, structure, and styles of a webpage.

How do you select an element from the DOM

To select an element from the DOM, JavaScript provides several methods:
1. document.getElementById(): Selects an element by its ID.
2. document.querySelector(): Selects the first element that matches a CSS selector.
3. document.querySelectorAll(): Selects all elements that match a CSS selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where you attach an event listener to a parent element instead of individual child elements. When an event is triggered on a child element, it propagates up (bubbling) to the parent, where the event listener is handled. This is useful for dynamically added elements or when there are many child elements.

How do you manipulate an element's attributes and styles using the DOM?

1. setAttribute(): This method is used to change an existing attribute or add a new one to an element. For example, you can change the source of an image or set a link's destination.
2. element.style.property: Each CSS property can be directly modified through JavaScript. For example, you can change colors, sizes, borders, etc.
3. classList.add(): Adds a new CSS class to the element.
